## Website Structure

This website will include the following structural elements:

### Common to all Pages

* Nav-bar on top, including:
- Brand symbol or name to link to home page.
- Link to shop.
- Link to blog. 
- Link to account login. 
- Link to about. 
* Footer, including:
- Social links.
- Contact details.
- Site language, e.g., copyright. 
- Accessibility policy.
